A propos

229 Cloud Tourism est realisé par une duo mixte d'etudiants développeurs dans le cadre de l'édition 2013 du Google Cloud Developper Challenge. Ce projet a été développe en PHP pour le cote serveur et l'utilisation des API, ainsi qu'en HTML5/Javascript pour les traitements coté client.
La particularité de 229 Cloud Tourisme est qu'il s'est ecoulé exactement une semaine entre l'idée initiale du concept, et la date de soumission. Toute l'application a été développee durant la dernière semaine du concours.
Notre equipe est composée de Armand E. R. M. ACCROMBESSI et G. C. Esperancia AHEHEHINNOU.

About 229CloudTourism

229 Cloud Tourism is made by a team of two students for the Google Cloud Developper Challenge 2013. This project uses PHP for the server-side and API code while client-side is operated with HTML5/Javascript.
An interesting fact about 229 Cloud Tourisme: it have been made exactly in one week, starting from November 14, 2013 (actually the last submission week). This includes the original concept idea and all the conception and coding.
Our team is composed of Armand E. R. M. ACCROMBESSI and G. C. Esperancia AHEHEHINNOU.
